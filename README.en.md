```text
QA-Glossary-Wiki/
|-- Sections/                 # English wiki terms (A-Z)
|-- Sections-zh/              # Chinese wiki terms (A-Z)
|-- scripts/                  # Core automation scripts
|   |-- scraper.py            # Scraper + HTML -> Markdown (with TOC)
|   |-- translate_articles.py # EN -> ZH translation pipeline
|-- state/                    # Runtime state and logs
|-- scraper.py                # Backward-compatible entrypoint
|-- translate_articles.py     # Backward-compatible entrypoint
|-- README.md                 # Chinese default README
|-- README.en.md              # English README
|-- LICENSE                   # MIT (scripts/code)
|-- LICENSE-CC-BY-NC-SA-4.0.md# CC BY-NC-SA 4.0 (glossary content)
```
