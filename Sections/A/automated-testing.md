# Automated Testing  自动化测试

自动化测试使用脚本来执行重复性任务，提高软件性能和测试效率。它提高了[测试覆盖率](../T/test-coverage.md)和执行速度，使[软件测试](../S/software-testing.md)过程更加有效。
