# Baseline Testing 基线测试

基线测试属于非功能测试的范畴，通过在特定条件下对系统或应用的性能或特质进行测量。这一初步的测量结果将作为未来性能评估的“基线”或对照标准。基线测试的核心目的是揭示系统当前的运作状态，并为后续的测试阶段建立一个性能衡量的标准。如果后续的测试结果与此基线存在偏差，那么可能意味着系统存在性能下降、功能退化或其他需要关注并解决的问题。
