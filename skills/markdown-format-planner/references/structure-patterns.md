# Structure Patterns

## Guide Pattern

1. `# Title`
2. `## Overview`
3. `## Prerequisites`
4. `## Steps`
5. `## Validation`
6. `## Troubleshooting`

## Reference Pattern

1. `# Title`
2. `## Summary`
3. `## Definitions`
4. `## Examples`
5. `## Notes`

## Update Pattern

1. `# Title`
2. `## Version or Date`
3. `### Added`
4. `### Changed`
5. `### Fixed`
