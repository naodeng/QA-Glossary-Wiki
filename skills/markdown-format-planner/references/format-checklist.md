# Format Checklist

- Single top-level heading (`#`) exists.
- Heading levels do not skip.
- One bullet style is used consistently.
- Blank lines are consistent and minimal.
- Code fences are closed and language-tagged when possible.
- Tables have valid header separators.
- Links use valid Markdown syntax.
- No trailing spaces.
- No duplicated empty lines.
- Content meaning is unchanged.
